# tickersymbol
Obtains quote information from a given stock symbol through terminal.

## Install
```
npm install -g tickersymbol
```

## Example Usage
```
tickersymbol aapl
```

outputs ...

```
Name:        Apple Inc.
Exchange:    NASDAQ
Last Trade:  5/8/2014 11:15am
Price:       592.49
Change:      +0.16
Change %:    +0.03%

Day Open     588.25
Day High     593.57
Day Low      588.25
Volume:      3269818
Day Avg Vol: 9931160
Year High:   604.41
Year Low:    388.87
Market Cap:  510.4B
```
