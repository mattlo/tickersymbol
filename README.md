# tickersymbol
Obtains quote information from a given stock symbol through terminal.

## Install
```
npm install -g tickersymbol
```

## Example Usage
```
tickersymbol aapl
```

outputs ...

```
Name:        Apple Inc.
Exchange:    NASDAQ
Bid:         591.42
Change:      -2.08
Change %:    -0.35%

Day Open     595.10
Day High     597.29
Day Low      587.73
Volume:      10102284
Day Avg Vol: 9931160
Year High:   604.41
Year Low:    388.87
Market Cap:  510.2B
```
